# E-ink Calendar
A low-power, ESP32-based desktop calendar that syncs with Google Calendar and monitors indoor air quality using an SCD41 sensor.

## Why I Made This
I made this project to keep track of my schedule and help me visualize what is happening in my day. I also wanted it to monitor readings in my room for example if the CO2 is getting too high or the temp is getting low I can see it in real time.
